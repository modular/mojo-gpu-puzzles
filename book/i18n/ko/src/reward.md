# 🏆 보상을 받아가세요

퍼즐을 모두 풀어보셨나요? 여러분의 도전을 축하하며 무료 스티커 팩을 선물로 드려요!

무료 스티커를 받는 방법:

1. GitHub 저장소 [https://github.com/modular/mojo-gpu-puzzles](https://github.com/modular/mojo-gpu-puzzles)를 Fork합니다
2. 퍼즐 솔루션을 작성해서 추가합니다
3. [이 양식](https://forms.gle/bchQpB3GanHMNY3x9)으로 제출하면 Modular 한정 스티커를 보내드려요!

현재는 **북미 지역**으로만 배송이 가능합니다. 다른 지역에 계신 분들도 솔루션을 제출해 주세요 – 배송 범위를 넓혀가고 있으니, 가능해지면 꼭 보상을 보내드릴게요.
